## Using CSS Variables

All theme values are exposed as CSS custom properties:

```css
.my-component {
  /* Colors */
  color: var(--duro-color-text);
  background: var(--duro-color-background);

  /* Spacing */
  padding: var(--duro-space-4);
  margin: var(--duro-space-2);
  gap: var(--duro-space-3);

  /* Typography */
  font-family: var(--duro-font-body);
  font-size: var(--duro-font-size-lg);
  line-height: var(--duro-line-height-normal);

  /* Visual */
  border-radius: var(--duro-radius-md);
  box-shadow: var(--duro-shadow-lg);
}
```
