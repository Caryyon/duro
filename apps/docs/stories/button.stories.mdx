## Design Tokens

Buttons use the following Duro design tokens:
- **Transform**: `translate(2px, 2px)` on hover for press-down effect
- **Shadow**: Hard offset shadows with 0 blur
- **Border**: 2px solid borders
- **Radius**: 0 (sharp corners)
- **Transition**: 0.2s ease for smooth interactions

## Usage

```tsx
import { Button } from '@duro/core'

// Primary button
<Button variant="primary" onClick={() => console.log('Clicked!')}>
  Click me
</Button>

// Destructive action
<Button variant="destructive" size="lg">
  Delete Account
</Button>

// Disabled state
<Button disabled>
  Processing...
</Button>
```
