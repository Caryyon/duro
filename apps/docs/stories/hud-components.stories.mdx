# HUD Components

A collection of HUD (Heads-Up Display) components for building tech interfaces, dashboards, and data visualizations.

---

## HudPanel

A styled container panel with corner brackets and optional status badge.

<Canvas>
  <Story name="HudPanel">
    <div style={{ background: 'var(--duro-color-background)', padding: '2rem' }}>
      <Stack gap="default">
        <HudPanel title="SYSTEM STATUS" status="NOMINAL">
          <p style={{ margin: 0, fontFamily: 'var(--duro-font-monospace)', fontSize: 'var(--duro-font-size-sm)' }}>
            Panel content goes here. This is a flexible container for any content.
          </p>
        </HudPanel>
        <HudPanel title="DATA STREAM" noPadding>
          <div style={{ background: 'var(--duro-color-muted)', padding: '1rem' }}>
            <p style={{ margin: 0, fontFamily: 'var(--duro-font-monospace)', fontSize: 'var(--duro-font-size-sm)' }}>
              Panel with noPadding prop - content fills the entire area.
            </p>
          </div>
        </HudPanel>
      </Stack>
    </div>
  </Story>
</Canvas>

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | required | Panel title displayed in header |
| `status` | `string` | - | Optional status badge text |
| `noPadding` | `boolean` | `false` | Remove padding from content area |

## StatBlock

A statistics display block with label, value, unit, and optional trend indicator.

<Canvas>
  <Story name="StatBlock">
    <div style={{ background: 'var(--duro-color-background)', padding: '2rem' }}>
      <div style={{ display: 'flex', gap: '1rem', flexWrap: 'wrap' }}>
        <StatBlock label="CPU LOAD" value="47" unit="%" trend={-3} />
        <StatBlock label="MEMORY" value="8.2" unit="GB" trend={12} />
        <StatBlock label="NETWORK" value="1.4" unit="Gbps" />
        <StatBlock label="UPTIME" value="127" unit="days" />
        <StatBlock label="THREADS" value="16" unit="" small />
      </div>
    </div>
  </Story>
</Canvas>

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | required | Label text displayed above the value |
| `value` | `string \| number` | required | Main value to display |
| `unit` | `string` | `''` | Unit suffix (e.g., "GB", "%", "ms") |
| `trend` | `number` | - | Trend percentage (positive = up, negative = down) |
| `small` | `boolean` | `false` | Use smaller size variant |

## Combined Example

Here's an example combining multiple HUD components into a panel:

<Canvas>
  <Story name="Combined Example">
    <div style={{ background: 'var(--duro-color-background)', padding: '2rem' }}>
      <HudPanel title="SYSTEM DIAGNOSTICS" status="ONLINE">
        <div style={{ display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '1rem' }}>
          <div>
            <DataRow label="Status" value="Operational" />
            <DataRow label="Uptime" value="127 days" highlight />
            <DataRow label="Load" value="47%" />
          </div>
          <div style={{ display: 'flex', flexDirection: 'column', gap: '0.5rem' }}>
            <div style={{ display: 'flex', justifyContent: 'space-between', alignItems: 'center' }}>
              <StatusIndicator status="active" label="PRIMARY" />
              <SignalStrength strength={5} />
            </div>
            <div style={{ display: 'flex', justifyContent: 'space-between', alignItems: 'center' }}>
              <StatusIndicator status="standby" label="BACKUP" />
              <SignalStrength strength={3} />
            </div>
          </div>
        </div>
        <div style={{ marginTop: '1rem' }}>
          <WaveformDisplay height={40} />
        </div>
      </HudPanel>
    </div>
  </Story>
</Canvas>
